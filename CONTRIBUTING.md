# Contributing

This repository is a **curated showcase**: design notes, process artifacts, and
safe screenshots—not the full production codebase.

- **Feedback on playtest builds:** use [Issues](https://github.com/umut-yasin-yilmaz/project-a-showcase/issues) with build tag / zip name and steps to reproduce.
- **Documentation fixes:** small typo or clarity fixes in `README.md` or `docs/*.md` are welcome via pull request.
- **Game code / design proposals:** not accepted here; source lives in a private development repository.

Thank you for helping keep the showcase accurate and readable.
