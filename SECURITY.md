# Security policy

## Scope

This repository is a **documentation and media showcase**. It does **not**
contain the full game source tree.

Playtest builds may be published as **GitHub Release** attachments (zipped
Windows exports). Treat downloaded zips like any other executable from the
internet: scan if your environment requires it, and run only from trusted
releases on this repository.

## Reporting a concern

If you believe you have found a security issue related to **release binaries**
or **repository configuration**, please report it responsibly:

- Prefer **[GitHub private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)** for this repository if enabled, **or**
- Open a non-public channel you use for maintainer contact (for example email), with enough detail to reproduce or assess impact.

Please do **not** file public issues that disclose exploitable details before a fix or maintainer response is available.
